Docker基础

发表于 2021-11-18 更新于 2022-07-28 分类于 1.环境部署 ， 1.其他 阅读次数：

安装Docker

脚本安装与手动安装

Docker教程：https://www.runoob.com/docker/docker-tutorial.html

Docker使用

docker 帮助

docker stats –help 单个参数（stats）的具体使用方法

docker pull ubuntu 下载镜像

docker run -it ubuntu /bin/bash 以交互方式运行docker（-i：交互式，-t：终端，ubuntu：镜像，/bin/bash：执行的命令）

docker ps 查看正在运行的容器

docker ps -a 查看所有存在的容器

docker start name/id 根据名字或id启动一个已停止的容器

docker run -itd …… -d：后台运行

docker stop name/id 停止一个容器

docker restart name/id 重启容器

docker attach name/id 进入在后台运行的容器（用该命令进入容器exit后会导致容器停止）

docker exec -it name/id /bin/bash 进入在后台运行的容器（用该命令进入容器exit后只会退出，而不会停止容器）

docker export name/id > ubuntu.tar 导出容器

cat docker/ubuntu.tar | docker import - test/ubuntu:v1 从容器快照导入容器

docker import http://example.com/exampleimage.tgz example/imagerepo 从URL或目录导入容器

docker rm -f name/id 强制删除容器

docker container prune 清理所有处于终止状态的容器

docker rename oldname newname 重命名容器

docker cp /路径/文件名 容器ID:/上传路径 传文件到docker

docker cp 容器ID:/上传路径 /路径/文件名 从docker传文件到本地

使用Docker创建一个WEB

docker run -d -p 8081:5000 –name webapp-test1 training/webapp python app.py 使用创建一个webapp并从容器内部的5000端口映射到主机的8081端口（-P不带参数可以随机映射端口）

docker port name/id 查看容器映射的端口信息

docker logs -f name/id 查看容器运行日志（-f标准输出，类似于tail -f）

docker top name/id 查看容器内部运行的进程

docker inspect name/id 检查容器底层信息，记录着容器的配置和状态，json格式

docker ps -l 查看最后一次创建的容器

Docker镜像

docker images 列出本机所有镜像

docker search 在网络上搜索镜像

docker rmi 删除镜像

docker commit -m=”has update” -a=”runoob” name/id runoob/ubuntu:v2 容器快照，将现有容器状态存为新的镜像（-m：提交的描述信息，-a：指定镜像作者，runoob/ubuntu:v2镜像名）

docker tag name/id xiaoluan/ubuntu:target 为镜像添加一个新的标签，一个镜像可以指定多个标签

使用Docker创建一个MSSQL数据库

docker pull mcr.microsoft.com/mssql/server:2017-latest 下载镜像

参考资料：https://www.jianshu.com/p/b1d9194900a2?utm_campaign=maleskine&utm_content=note&utm_medium=seo_notes&utm_source=recommendation

docker run -e “ACCEPT_EULA=Y” -e “SA_PASSWORD=<YourPassword!>” -p 8001:1433 –name mssql1 -d mcr.microsoft.com/mssql/server:2017-latest 运行mssql的docker容器

docker exec -it mssql1 /opt/mssql-tools/bin/sqlcmd -S localhost -U SA -P “<YourPassword!>” -Q ‘ALTER LOGIN SA WITH PASSWORD=”<YourPassword!>”‘ 更改旧的SQL SERVER的SA密码

docker exec -it mssql1 “bash” 与mssql容器交互

/opt/mssql-tools/bin/sqlcmd -S localhost -U SA -P “<YourNewStrong!Passw0rd>” 调用sqlcmd登入mssql
